# Codex Web UI Pack: Constitution

## When to use
- Beginning a new repo or hardening project principles.
- `.specify/memory/constitution.md` is missing or outdated.

### COPY/PASTE INTO CODEX WEB TASK
```text
Phase: Constitution only.

Use templates/commands/constitution.md as canonical behavior reference.
Create or update .specify/memory/constitution.md with project principles and governance.

Rules:
- Never edit .specify/templates/* except via specify init/upgrade.
- Do not create specs/* feature content in this phase.
- Do not implement code in this phase.

Output:
1) Proposed constitution diff
2) Brief rationale for each principle change
3) STOP after presenting diff
```
